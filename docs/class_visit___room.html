Detailed Description

This class defines the Visit_Room state.

This class implements the Visit_Room state of the final state machine, it receives the room chosen by 'Plan_To_Urgent' state. It calls the method for sending the goal position to the move_base client, then calls the the function to simulate the robot visiting the location if it's a room, by doing a rotation of 360 degrees. Then returns to 'Plan_To_Urgent' This is made always checking in every loop the state of the battery to see if robot has to go recharging.
