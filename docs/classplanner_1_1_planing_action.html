This function is the callback of the robot's planner class.

Parameters
goal
Returns
None

The callback invoked when a client set a goal to the planner server. This function will return a list of random points (i.e., the plan) when the fist point is the current robot position (retrieved from the my_helper node), while the last point is the goal position (given as input parameter). The plan will contain a random number of other points, which spans in the range [self._random_plan_points[0], self._random_plan_points[1]). To simulate computation, each point is added to the plan with a random delay spanning in the range [self._random_plan_time[0], self._random_plan_time[1]).
